# WeTreeOS（自己动手写操作系统）

* 背景：

  * 假期无聊
  
  * 对于操作系统的很多概念没有掌握

---

* 开发环境

  * 汇编器：NASM

  * 编译器：GCC

  * 反汇编工具：objconv [GitHub地址](https://github.com/vertis/objconv)

  * 写盘工具：Java自制写盘工具 （Linux下dd也可）

  * 系统环境：

    * Windows10-1909

    * Ubuntu14.04-LTS-i386(32位)

  * 虚拟机：

    * Oracle Virtual Box

    * QEMU（改变分辨率后存在问题）
